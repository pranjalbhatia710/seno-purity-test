# The Seno Purity Test

A personal Rice Purity Test-style checklist for everyone in Seno lore.

## Live site

https://pranjalbhatia710.github.io/seno-purity-test/

## Current format

- Continuous 67-item checklist with no category segregation.
- Score displays as `67/67` and decreases as items are checked.
- Checking question 67 intentionally blanks the score and triggers a screen shake.
- No emoji UI.
- 67 generated per-question SVG images in `assets/generated/`.
- Actual Seno photos from the shared album are used in the header and generated-image backgrounds.
- Static HTML/CSS/JS, deployed on GitHub Pages.

## Secret note

Do not commit or paste API keys. If external image generation is needed, configure a rotated key locally via environment variables outside chat/tool logs.
